## Semantic Analysis of Spotify Song Lyrics and User Mood Trend

## Project Overview
This project performs sentiment and emotion analysis on Spotify song lyrics using NLP techniques in Python.

## Features
- Sentiment Analysis
- Emotion Detection
- Word Cloud Visualization
- Artist Emotion Comparison
- Mood Trend Analysis

## Technologies Used
- Python
- Pandas
- Matplotlib
- TextBlob
- WordCloud
- Google Colab

## Results
The project identifies emotional patterns and mood trends in song lyrics using data visualization and NLP.
